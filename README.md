### Offline Documentation

Clone or Download, then goto download root dir

Install dependencies

```bash
composer install
```

Configure your environment

```bash
cp .env.example .env
php artisan key:generate
```

Serve the docs

```bash
php artisan serve
```

Enjoy reading the documentation...
